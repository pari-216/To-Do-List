# FlowTasks - Modern Task Management Application

A beautiful, responsive, and feature-rich to-do list application built with vanilla HTML, CSS, and JavaScript. Perfect for an internship portfolio that showcases professional frontend development skills.

## ✨ Features

### Core Functionality
- ✅ **Local Storage Persistence** - Tasks are automatically saved and restored on page refresh
- ✅ **Task Management** - Add, edit, complete, and delete tasks seamlessly
- ✅ **Task Priorities** - Assign High, Medium, or Low priority levels with visual indicators
- ✅ **Smart Filtering** - Filter by All, Pending, or Completed tasks
- ✅ **Bulk Actions** - Clear all completed tasks with one click

### User Interface
- 🎨 **Modern Card-Style Layout** - Clean, contemporary design with glassmorphism effects
- 📱 **Fully Responsive** - Optimized for mobile (≤480px), tablet (480-768px), and desktop (≥768px)
- 🌙 **Dark Mode** - Toggle between light and dark themes with system preference detection
- ✨ **Smooth Animations** - Task appearance, completion feedback, and hover effects
- 🎉 **Celebration Effect** - Confetti animation when all tasks are completed
- 📊 **Statistics Dashboard** - Real-time display of total, completed, and pending tasks
- 📈 **Progress Bar** - Visual representation of task completion percentage
- 📅 **Date Display** - Shows today's date in the header

## 🚀 Getting Started

### Installation
1. Clone or download the repository
2. Open `index.html` in your web browser
3. Start adding tasks!

### Usage
- **Add Task**: Type task, select priority, click "Add" or press Ctrl+Enter
- **Complete Task**: Click the checkbox next to a task
- **Edit Task**: Click the edit button (pencil icon)
- **Delete Task**: Click the delete button (trash icon)
- **Filter Tasks**: Use filter buttons for All, Pending, or Completed
- **Clear Completed**: Click "Clear" to remove all completed tasks
- **Toggle Theme**: Click the moon/sun icon in the header

### Keyboard Shortcuts
- `Ctrl+Enter` / `Cmd+Enter` - Add task
- `Escape` - Clear task input
